# Web programming project

A file storage web application similar to Google Drive, DropBox etc. done for the purpose of an Intro to Web programming course at the <a href="https://www.pmf.uns.ac.rs/en/"> Faculty of Sciences, Novi Sad, Serbia</a>. 

The project is done in the PHP programming language :elephant:.

## Things implemented:
A user can log in to their acc and a list of directories and files of their root directory are shown. 
A user can then go through directories and add new ones to the current. 
A user can upload a new .txt, .png, .jpeg/.jpg file to the current directory.
A user can also open the files and go to the last 6 opened files at all times.
